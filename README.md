#Ejecutar en desarrollo

1.-Clonar repositorio

2.-Ejecutar

```
npm install
```


3.- Tener NestJs CLI Instalado

```
npm install -g @nestjs/cli
```

4.-Levantar BBDD
```
docker-compose up -d
```
